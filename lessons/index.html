Node.js 后端之路

从前端工程师 → 能独立上线收费 SaaS 的全栈开发者。
贯穿项目：LinkPulse（短链接 + 访问分析 + 订阅付费）。

🧭 MISSION（学习指南针） 📖 RESOURCES（可信资源）

课程（Lessons）

    01

    Node 的本质与执行顺序 ✓ 已完成

    单线程 + 非阻塞 I/O、事件循环、同步→微任务→宏任务的执行顺序。阶段 1。

         02

    事件循环内部机制（图解） ✓ 已完成

    调用栈、各队列存在哪、事件循环 6 阶段；附 A E C 666 B 的可点击逐步动画。阶段 1。

         03

    HTTP 基础与你的第一个服务器 ✓ 已完成

    HTTP 请求/响应解剖、用原生 http 写服务器、req/res、路由（routing）。阶段 1。

         04

    模块系统与第一个正式项目 ✓ 已完成

    ESM import vs CommonJS require、package.json、npm scripts、真实项目结构。阶段 1。

         05

    用 Express 重写服务器 ✓ 已完成

    第一次 npm install 真实依赖；app.get 路由、res.json、路由参数 :code，告别 if/else。阶段 2。

         06

    中间件与处理 POST 请求体 ✓ 已完成

    middleware 概念与执行顺序、express.json() 解析 req.body、写日志中间件、POST 创建短链雏形。阶段 2。

         07

    输入校验与错误处理 下一课 →

    用 zod 校验 req.body、统一错误处理中间件、「永不信任用户输入」。阶段 2。

         08

    路由拆分与分层架构 待学

    express.Router、route / controller / service 三层、关注点分离。阶段 2。

         09

    配置管理与结构化日志 待学

    .env / dotenv、pino 结构化日志、12-factor 配置、密钥不进 git。阶段 2。

         10

    给项目加上 TypeScript 待学

    tsconfig、tsx、类型化 req/res、TS 基础补齐。此后代码全用 TS。阶段 2。

         11

    数据库入门与 PostgreSQL 待学

    为何用数据库、关系型 vs 文档型、用 Docker 起 Postgres 并连接。阶段 3。

         12

    SQL 基础 待学

    建表、增删改查、约束（PK/FK/UNIQUE/NOT NULL）、索引初步。阶段 3。

         13

    数据建模：为 LinkPulse 设计表 待学

    ER 建模、主外键、1NF→3NF、索引/迁移、设计 users / links / clicks 三表。阶段 3。

         14

    Prisma 入门 待学

    ORM 是什么、schema.prisma、prisma migrate、类型安全查询。阶段 3。

         15

    用 Prisma 实现真正的短链 CRUD 待学

    nanoid 生成短码、唯一冲突处理、分页、真正把短链持久化进数据库。阶段 3。

         16

    短链跳转 + Redis 缓存 待学

    res.redirect 302、cache-aside 缓存模式、ioredis。LinkPulse 核心闭环达成 🎉。阶段 3。

         17

    认证基础：注册 / 登录 / JWT 待学

    认证 vs 授权、密码哈希（bcrypt）、JWT 签发，给 LinkPulse 加注册登录。阶段 4。

         18

    授权与 Web 安全加固 待学

    鉴权中间件、越权（IDOR）防护、RBAC、OWASP Top 10、helmet、限流。阶段 4。

         19

    设计模式与 SOLID 待学

    依赖注入、Repository 模式、SOLID 原则，重构 LinkService 让其可测可换。阶段 4。

         20

    后台任务与队列（BullMQ） 待学

    点击异步入队、独立 worker 落库、重试/幂等、定时聚合任务。阶段 5。

         21

    实时点击看板（WebSocket / SSE） 待学

    轮询 vs SSE vs WebSocket 选型、用 ws 广播、worker 推送实时点击。阶段 5。

         22

    性能优化与压测 待学

    autocannon 压测、N+1 问题、索引、连接池、不阻塞事件循环。阶段 5。

         23

    自动化测试 待学

    测试金字塔、Vitest 单元（注入 mock）、supertest 集成测 API。阶段 6。

         24

    用 Docker 容器化 待学

    多阶段构建编译 TS、.dockerignore、docker-compose 一键起 app+pg+redis。阶段 7。

         25

    CI/CD 与部署上线 待学

    GitHub Actions（test+build）、部署到 Render、migrate deploy、Sentry 监控。阶段 7。

         26

    Stripe 订阅计费 + 上线清单 收官 🎉

    Stripe Checkout + Webhook、套餐限额、上线清单。LinkPulse 成为可收费 SaaS！阶段 8。
